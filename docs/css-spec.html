CSS 规范

概述

规范的目的在于使我们的CSS代码更加易于维护和重用，从而提升效率。

选择器命名
  • 命名除 .fn- / .ui- / .sl- 外，可自定义命名。命名以简明扼要为原则。
  • 命名包含多个单词时，使用 “-” 连接，如果 .ui-pop 。
  • 一般情况下，如果命名比较通用，比如 .current ，请限定在相应的上下文环境中。建议写成 #parent .current 。
  • 作为 JS 接口的 class 和 ID ，必须以 .js- 开头。
选择器书写

多于2个的选择器使用同样的属性时，每个各占一行。

.selector1,
.selector2,
.selector3,
...
.selectorN { property: value; }

避免使用低效的选择器。详见：https://developers.google.com/speed/docs/best-practices/rendering

body > * {...}
div#ID {...}
div#ID > h3 {...}
#ID span.class a {...}
div:last-child {...}
属性书写

不强制属性书写顺序，但推荐按照 “显示属性 – 自身属性 – 文本属性 – 其他属性(hack)” 的书写，遵循因果关系。

/* 显示属性 */            /* 自身属性 */            /* 文本属性 */
display                  width                    color
list-style               height                   font
position                 margin                   text-decoration
float                    padding                  text-align
clear                    border                   vertical-align
z-index                  background               white-space
                                                  other text
                                                  content

浏览器私有属性在前，标注属性在后

-webkit-border-radius: 5px;
   -moz-border-radius: 5px;
    -ms-border-radius: 5px;
     -o-border-radius: 5px;
        border-radius: 5px;
文件书写

要有全局设置

/* global set */
body { font: 14px/1.5 Arial, \5FAE\8F6F\96C5\9ED1; ... }
a { color: #458FCE; text-decoration: none; }
a:hover { text-decoration: underline; }

减少直接在标签上定义样式，请使用类。 

/* Bad */
h1 { font-size: 22px; }
h2 { font-size: 20px; }
h3 { font-size: 18px; }

/* Good */
.txt-big { font-size: 22px; }
.txt-normal { font-size: 20px; }
.txt-small { font-size: 18px; }
浏览器 Hack

使用以下形式区别浏览器。

/* 区别属性 */
IE6                       _property: value;
IE6/7                     *property: value;

/* 区别规则 */
IE6                       * html selector { … }
IE7                       * :first-child+html selector { … }
not IE6                   html > body selector { … }
firefox only              @-moz-document url-prefix() { … }
saf3+/chrome1+            @media all and (-webkit-min-device-pixel-ratio:0) { … }
opera only                @media all and (-webkit-min-device-pixel-ratio:10000),
                          not all and (-webkit-min-device-pixel-ratio:0) { … }
iPhone/mobile webkit      @media screen and (max-device-width: 480px) { … }

使用更优雅的 Hack

.selector .child { property: value; } /* for ie6 */
.selector > .child { property: value; } /* except ie6 */
注释

在文档的顶部声明组件名，组件介绍，依赖的样式，作者。

/**
 * @name: 组件名
 * @overview: 组件介绍
 * @require: 依赖的样式
 * @author: 作者
 */

按功能模块注释，在模块的结尾空一行表示结束。

/* news list */
.list li {...}
.list li a {...}

/* gallery list */
.gallery li {...}
.gallery li a {...}
